# BlogApi Frontend
A frontend of the Blog project.


## Setup

```
 Rename .env.renameMe file to .env
```

## Run the application for development

```
$ npm start
```

- [Mongoose](https://mongoosejs.com/)

- [React](https://reactjs.org/)

- [Redux](https://redux.js.org/)
